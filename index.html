收件流程及注意事項

星鴻包租代管－彰一處 內部進件防呆指南

第一大類：直媒案件 / 間媒案件
間媒案件留意事項 & 信任租話術
  • 主動告知義務： 帶看時，必須主動告知房客「會調閱信用狀況提供給房東評估」。盡量調閱以進行租客篩選，有利於後續管理方便。
  • 說明信任租話術懶人包 (業務可直接背誦口條)：
    我們會需要調閱您的信用狀況，一來可以讓房東更加了解您的信用狀況，但是請放心，我們調閱的結果是不會傳給房東看，只會口頭告知評等是好、尚可、不好，主要針對的是您在信用方面的狀況是否有不良紀錄，而且這不會扣你的聯徵分數，若您沒問題麻煩幫我在這張同意書上簽名及身分證正反面傳給我。
  • 確認承租意願： 在帶看的同時，請務必確認房客的「租期意願」。若房客明確表達願意長期承租，業務可直接與房東討論租期部分是否可以直接簽 三年期。
包租案件必須收集資料
  • 1. 房客信任租調閱：
    • 必須收齊「身分證正反面 + 同意書」後，回傳至信任租調閱群組。
    • 時間關鍵 每日 10點前 送出，可在當天知道結果。業務須隨時留意主管是否已協助送出。
    • 審核硬限制 信任租評等在 C以下 (含C) 需額外上簽呈 (此限制僅包租案需要)。
  • 2. 六個月繳租證明：
    • 必須使用銀行轉帳證明，絕對不可使用簽收單、手寫收據。
    • 若無法提供：必須提供保證人，且保證人與房客要一同辦理公證 + 簽呈 (保證人同樣要提供信任租審查)。
    • 若缺少月份：業務必須上簽呈。至於是否一定要保證人？「請當下直接與主管確認！」
  • 3. 三名緊急聯絡人：
    • 請房客務必完整提供 3名 緊急聯絡人資訊，關係必須包含：家人、朋友、同事 各一。
    • 話術安撫：告知房客平常絕對不會打電話，除非發生極為緊急狀況需要聯繫時才會主動聯繫。
    • 防呆 緊急聯絡人資料必須確實於房客系統資料上建立留檔。
第二大類：包租案件 / 代管案件
代管案件注意事項 & 簽呈範本
  • 代管一年規定：若案件為「代管一年案」，業務必須上簽呈給主管。
  • 複審夾檔：簽呈核准後，主管會截圖核准畫面。業務必須將此截圖夾帶在複審資料中，位置請放置在「協議書之後」即可。
  • 📋 中區代管簽呈範本：
    主旨：中區代管簽呈 說明： 一、案件資料 物件地址： 物件房型： 物件租金： 業務／代管件數： 房客工作職業： 二、案件說明 （一）本案為代管新媒合案件。 （二）檢附信任租結果：通過。 （三）房客原規劃可簽訂長期租約，惟因初次入住該社區，考量居住適應問題，經房東與房客雙方討論後，先行簽訂一年期租約；後續如續約評估狀況良好，雙方將再行商討長約簽訂安排。 懇請協理核准此代管案件。
包租 / 代管【房東】所需資料與 LINE 訊息
  • 1. 身分證正反面： 必須完整拍到四個邊角且清晰。可在上面壓字但絕不能遮擋到重要資訊及照片。建議壓字三擇一：星鴻社會專用、星鴻社會住宅及調閱課稅明細使用、租賃使用。注意 若要代辦調閱課稅明細，身分證上必須有壓到相關字眼才能代辦。
  • 2. 收租存摺封面： 必須完整拍到四個邊角，且必須清晰顯示出「分行名稱」、「帳號」和「戶名」。
  • 3. 房屋稅單或課稅明細： 必須提供當年度最新的版本。
  • 4. 第一類建物謄本： 用於核對建物的主要用途是否符合規定。
  • 業務代辦備註 第3項(稅單)及第4項(謄本)可由業務協助代辦，但必須提早跟房東說明清楚。
  • 特殊狀況補充：
    • 新建案/買賣交屋： 房東初期無法取得房屋稅單，業務須引導或協助至當地稅務局申請「公文函」，申請時間約需 1週 左右。
    • 稅率錯誤(營業稅/非住非營)： 若稅單上顯示此類稅率，須請房東親自至國稅局辦理變更為住家用稅率，辦理時間約需 7-14天 左右。
  • 💬 傳給房東的 LINE 資料收集訊息範本：
    房東您好再麻煩您幫我準備以下資料 1.([)房東(])彩色身分證正反面照片 2.([)房東(])彩色存摺照片（申請修繕補助費用用） 3.第一類建物謄本（有效期一個月內、可代辦） 4.房屋稅繳款書或課稅明細(可代辦) 房東若今年度房屋稅單找不到的話，我這邊可以協助辦理，一類謄本部分我會直接幫您代辦，先跟您說一聲。若證件上須要押上文字，請幫我壓「星鴻社會專用」或「星鴻社會住宅及調閱課稅明細使用」即可。
包租 / 代管【房客】通用資料與一般戶狀況
  • 1. 身分證正反面： 須完整拍到四邊且清晰，可在上面壓字但不能遮擋到重要資訊及照片，壓字三擇一：星鴻社會專用、星鴻社會住宅及調閱課稅明細使用、租賃使用。
  • 2. 領取300億租金補貼帳戶： 須完整拍到四邊，清晰顯示分行名稱、帳號和戶名。
  • 3. 戶籍謄本或戶口名簿： 必須為「全戶含詳細記事」。戶籍謄本須在 近一個月內 申請；戶口名簿則須上網查詢是否有異動紀錄並截圖證明。
  • 4. 外籍配偶： 必須額外提供居留證正反面及護照基本資料頁。
  • 一般戶租客常見狀況：
    • 使用非本人帳戶： 須加附指定帳戶人的身分證正反面，且簽約時須加簽「金融帳戶變更切切書」。
    • 絕對禁用 不可使用無實體銀行的網銀帳戶 (例如：將來銀行、王道銀行、LINE Bank等一律退件)。
  • 關鍵補充 (攸關業績獎金)：
    租客若提前入戶籍，記得在 一個月內 完成社宅合約才視為新件！且必須要在 【表五 承租人申請書 P2】的戶籍地址後方親筆加上「遷入時間」。若未加上，審查一律視為舊件，獎金將會被砍半！
  • 💬 傳給房客的 LINE 資料收集訊息範本：
    房客您好再麻煩您幫我準備以下資料 1.([)房客(])彩色身分證正反照片 2.([)房客(])存摺照片 3.戶籍謄本([)全戶含詳細記事且近一個月內(]) 若證件上須要押上文字，請幫我壓「星鴻社會專用」或「星鴻社會住宅及調閱課稅明細使用」即可。
⚠️ 【必看】弱勢戶租客身分認定與防呆
  • 核心硬性要求 弱勢戶進件必須提供有效的弱勢證明文件，並依下列分類嚴格核對：
  • 1. 育有未成年子女 (父母與小孩不同戶籍)：
    • 業務必須額外去申請小孩的戶籍謄本，此部分接受申請「部分限戶」即可過關。
  • 2. 懷孕第二胎 (目前為胎兒身分)：
    • 審查重點 必須精準拍攝孕婦媽媽手冊的「正面封面、反面封底、以及內頁有醫院蓋章的產檢紀錄」。
    • 審查時會嚴格核對產檢紀錄日期是否為最新日期。
    📸 經理可在代碼此處更換為孕婦手冊範本圖片
  • 3. 身心障礙身分：
    • 必須提供身障小卡的正反面照片。
    • 防呆 業務務必當場核對小卡背面的「有效期限」是否過期。
  • 4. 低收入戶 或 中低收入戶：
    • 直接提供最新年度的證明書。
    • 注意誤區 市面上有一種證明文件是「中低收老人」，其核定規格不同，切勿搞混。
  • 5. 特殊境遇家庭：
    • 必須提供正式的「特殊境遇家庭身分證明文件」。
    • 絕對退件 不可以提供公所的核定函，一律不予承認。
  • 6. 滿60歲高齡戶 或 原住民身分：
    • 不需要額外準備複雜證明，只需確認檢附的戶籍謄本（全戶含詳細記事）上有明確紀載出生年月日或原住民身分註記即可。
